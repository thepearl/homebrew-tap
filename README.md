# Homebrew Tap for thepearl

This is a Homebrew tap containing formulas for thepearl's projects.

## Installation

```bash
brew tap thepearl/tap
brew install xcode-janitor-mcp
```

Or install directly:
```bash
brew install thepearl/tap/xcode-janitor-mcp
```

## Available Formulas

- **xcode-janitor-mcp** - MCP server for cleaning unused assets in Xcode projects
  - [Homepage](https://github.com/thepearl/xcode-janitor-mcp)
  - [Documentation](https://github.com/thepearl/xcode-janitor-mcp#readme)
